Instructions for Backpage Header: This pattern is on all back pages by default. To change the background image, upload the photo you wish to use, then either label the photo with backpage-background (if you want it to show on all backpages), or label the photo with the page slug and a -header-background suffix. For example, if you want a unique background for a particular page, and the page is called “About Us,” label the file about-us-header-background.
